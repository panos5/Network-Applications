  • Overall design and site functionality:

    • This website is an online photo competition. The user has the option to upload one photo each week and likes photos from other users as well. At the end of the week the winner is the user whose photo gathered most of the likes. The theme of the competition changes every Sunday.

    • The new features of HTML5 and CSS3 that we have applied are canvasses to show the photos, transitions and animations in the main screen and drag-drop option to the upload page. Also the main graphics of the site follow tha window size(after refresh).

    • We have developed this website using Mozilla-Firefox(35.0.1).The site is also compatible with Google Chrome(40.0) and Opera(12.16).
